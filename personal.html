Nucleo S.A. - Personal

Proyecto: Plataforma Cloud & Operaciones en Google Cloud

Rol: DevOps / Cloud Engineer

Núcleo S.A. ees una empresa de telecomunicaciones y servicios digitales, donde el producto principal se apoya en una infraestructura cloud altamente disponible, escalable y segura para soportar aplicaciones críticas y grandes volúmenes de tráfico.

El ecosistema tecnológico está construido sobre Google Cloud Platform (GCP), integrando servicios de contenedores, cómputo administrado, redes, bases de datos y herramientas de observabilidad, con el objetivo de garantizar continuidad operativa, rendimiento y crecimiento sostenido.

CEO

Rol dentro del proyecto

Como DevOps Cloud Engineer, soy responsable del diseño, implementación y operación de la infraestructura en Google Cloud, actuando como nexo entre desarrollo, operaciones y negocio.

Contenedores y Orquestación

  • Diseño, despliegue y administración de clústeres Google Kubernetes Engine (GKE) para workloads productivos.
  • Implementación de aplicaciones serverless mediante Cloud Run y App Engine Flexible, optimizando costos y escalabilidad.
  • Gestión del ciclo de vida de imágenes de contenedores utilizando Artifact Registry, asegurando versionado y trazabilidad.

Balanceo de carga y gestión de tráfico

  • Configuración de Cloud Load Balancing para garantizar alta disponibilidad y distribución eficiente del tráfico.
  • Implementación de reglas de enrutamiento avanzadas y terminación SSL.
  • Gestión de certificados SSL para entornos productivos y no productivos.

Monitoreo y Observabilidad

  • Implementación y administración de Cloud Monitoring y Cloud Logging.
  • Creación de dashboards personalizados, métricas clave y alertas proactivas para detección temprana de incidentes.
  • Uso de Cloud Trace y Cloud Profiler para análisis de latencia, rendimiento y optimización de aplicaciones.

Bases de datos y Big Data

  • Administración de bases de datos relacionales con Cloud SQL y NoSQL con Cloud Firestore.
  • Diseño y optimización de soluciones analíticas utilizando BigQuery para procesamiento y análisis de grandes volúmenes de datos.
